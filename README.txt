ІНСТРУКЦІЯ ДЛЯ ЗАПУСКУ ПРОЄКТУ

Тема:
Автоматизована інформаційна система "Місця проведення дозвілля громадян"

Файли проєкту:
1. index.html      - головна сторінка
2. objects.html    - список діючих об'єктів дозвілля
3. events.html     - заходи на найближчі 14 днів
4. search.html     - пошук об'єктів за типом
5. style.css       - дизайн сайту
6. script.js       - JavaScript і підключення до Supabase
7. supabase.sql    - створення таблиць і тестові дані

ЩО РЕАЛІЗОВАНО
1. Головна сторінка з описом системи.
2. Список діючих об'єктів:
   - тип;
   - назва;
   - адреса.
3. Список заходів на найближчі 14 днів:
   - дата;
   - назва заходу;
   - назва об'єкта;
   - адреса.
4. Пошук об'єктів за типом.
5. Адаптивний сучасний дизайн.
6. Дані завантажуються із Supabase через API.
